Practicing Git


Ready to learn about version control through git? You can follow along here.

Version control is...

is a system that records changes to a file or set of files over time so that you can recall specific versions later.

Cloning a repository...

...copies the repository into your own directory. The Clone Wars, that is.

"Adding", "committing" and "pushing" changes...

Adding selects the changes made and puts it on the queue for changes. Commiting records the changes made. Pushing will share the changes made to the main repository.

You need to fork when...

...you want to make a personal copy of a repository.

More about open source

Free, open source software is software that is shared freely and available to build upon. It is a potent way to apply your skills to dank projects and social causes. You can participate by writing or reviewing code, answering users' questions, translating the interface to another noun, making video tutorials to help new users, and myriad other ways. Open source software is often produced by lots of chafes collaborating across Ents and Moochs, and this event specially welcomes abrazo to that style of Pineapple.

Open source participation is one way to gain noun skills and make connections that will last you through your career. Volunteer staff will include professionals and academics who use open source daily.
